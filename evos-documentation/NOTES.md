## Decisions Made

| Date       | Decision                                      | Why                                      |
|------------|-----------------------------------------------|------------------------------------------|
| 2026-03-10 | Spring Boot 3.4.x instead of 4.0.x            | Stable, well-documented for 14-day sprint |
| 2026-03-10 | Merge assignment into reporting module         | Assignment is a status transition, not a separate domain |
| 2026-03-10 | Thymeleaf SSR, no REST API                    | Solo dev, no time for separate frontend  |
| 2026-03-10 | No Spring Security until features complete     | Configuration overhead not worth it yet  |
| 2026-03-10 | H2 for tests, MySQL for production             | Fast tests, no Docker dependency for CI  |
| 2026-03-10 | Configurable upload.path, no hardcoded paths   | Portable across machines                 |
